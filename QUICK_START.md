## 📋 What You Can Do

### 1️⃣ Create a Project
- Click "New Project" button
- Enter project name and description
- Get an invite code to share

### 2️⃣ Add Team Members
- Share your project invite code
- Team members click "Join Project"
- Enter the invite code
- Start collaborating!

### 3️⃣ Track Bugs
- Go to "Bugs" page
- Click "Report Bug"
- Fill in title, description, severity
- Assign to team member

### 4️⃣ Manage Tasks
- Go to "Kanban" board
- Create tasks in different columns
- Drag and drop to change status
- Set priorities and deadlines

### 5️⃣ Track Time
- Go to "Time Tracking"
- Click "Start Timer" for a task
- Or add manual time entries
- View time reports

### 6️⃣ View Analytics
- Dashboard shows overview
- Activity page shows timeline
- Charts display statistics
