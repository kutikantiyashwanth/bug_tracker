# User Guide

## Getting Started

### 1. Register
Go to https://bug-tracker-ui-evqv.onrender.com/register
- Enter your name, email, password
- Select your role: Admin, Developer, or Tester
- Add your skills (optional)
- Click Create Account

### 2. Login
Go to https://bug-tracker-ui-evqv.onrender.com/login
- Select your role
- Enter email and password
- Click Sign In

---

## Admin Guide

### Create a Project
1. Go to Projects in the sidebar
2. Click New Project
3. Enter name and description
4. Click Create Project
5. Copy the invite code from the project card

### Share Invite Code
- The invite code appears on the project card (admins only)
- Click Copy Invite Code
- Share the UUID with your team members

### Manage Tasks
1. Go to Kanban Board
2. Click + in any column to create a task
3. Fill in title, description, priority, assignee, due date
4. Drag cards between columns to update status

### Manage Bugs
1. Go to Bug Reports
2. Click Report Bug
3. Fill in all details
4. Use AI Analyze to auto-detect severity
5. Assign to a developer

---

## Developer Guide

### Join a Project
1. Go to Projects page
2. Click Join Project
3. Enter the invite code from your Admin
4. Click Join Project

### Work on Tasks
1. Go to Kanban Board
2. Find tasks assigned to you
3. Drag to In Progress when you start
4. Drag to Testing when ready for review
5. Admin moves to Done when complete

### Fix Bugs
1. Go to Bug Reports
2. Find bugs assigned to you
3. Click on a bug to open details
4. Add comments to communicate progress
5. Click Mark as Resolved when fixed

---

## Tester Guide

### Join a Project
1. Go to Projects page
2. Click Join Project
3. Enter the invite code from your Admin

### Report a Bug
1. Go to Bug Reports
2. Click Report Bug
3. Enter title and description
4. Add steps to reproduce
5. Upload a screenshot (optional)
6. Click AI Analyze for severity suggestion
7. Click Submit Bug Report

### Track Bug Status
- Open: Bug reported, not yet assigned
- In Progress: Developer is working on it
- Resolved: Developer marked as fixed
- Closed: Admin confirmed and closed

---

## Notifications

Notifications appear in the bell icon in the top bar.

Types:
- Task Assigned — a task was assigned to you
- Bug Assigned — a bug was assigned to you
- Bug Resolved — a bug you reported was fixed
- Deadline Reminder — a task is due soon

Click the bell to view all notifications.
Click Mark All Read to clear the badge.
