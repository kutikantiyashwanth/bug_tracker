## 📝 Important Notes

### Keep Servers Running
Both frontend and backend servers need to stay running for the website to work.

### Stop Servers
If you need to stop the servers, press `Ctrl+C` in the terminal where they're running.

### Restart Servers
If servers stop, run:
```bash
# Frontend
cd frontend
npm run dev

# Backend
cd backend
npm run dev
```
