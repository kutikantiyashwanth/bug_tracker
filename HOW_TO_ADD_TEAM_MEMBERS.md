# 👥 How to Add Team Members

## 📋 Quick Guide

There are **two ways** to add team members to your project:

---

## Method 1: Share Invite Code (Recommended)

### For Project Owner/Admin:

1. **Find Your Invite Code**
   - Go to **Projects** page in the sidebar
   - Look at your project card
   - You'll see an invite code like: `CC-2026-XK9`
   - **Click the invite code** to copy it to clipboard
   - Or find it in the **sidebar** under your project name

2. **Share the Code**
   - Send the invite code to your team members via:
     - Email
     - Slack/Discord
     - WhatsApp/Telegram
     - Any messaging platform

### For Team Members:

1. **Register an Account**
   - Go to the website
   - Click "Sign up"
   - Fill in your details (name, email, password, role)
   - Click "Create Account"

2. **Join the Project**
   - After logging in, go to **Projects** page
   - Click **"Join Project"** button (top-right)
   - Enter the invite code you received
   - Click **"Join Project"**
   - ✅ You're now part of the team!

---

## Method 2: Direct Registration with Code

### Step-by-Step:

1. **Team Lead Creates Project**
   ```
   ✅ Login to the app
   ✅ Go to Projects page
   ✅ Click "New Project"
   ✅ Enter project name and description
   ✅ Click "Create Project"
   ✅ Copy the invite code (e.g., MY-2026-ABC)
   ```

2. **Team Lead Shares Code**
   ```
   Send to team members:
   "Join our project using code: MY-2026-ABC"
   ```

3. **Team Members Register**
   ```
   ✅ Go to website
   ✅ Click "Sign up"
   ✅ Fill in registration form
   ✅ Click "Create Account"
   ```

4. **Team Members Join**
   ```
   ✅ Go to Projects page
   ✅ Click "Join Project" button
   ✅ Enter invite code: MY-2026-ABC
   ✅ Click "Join Project"
   ✅ Done! Now you can see the project
   ```

## ❓ Troubleshooting

### Problem: "Invalid invite code"

**Solutions:**
- ✅ Check if code is typed correctly (case-sensitive)
- ✅ Make sure there are no extra spaces
- ✅ Verify the code with team lead
- ✅ Code format should be: `XX-YYYY-ZZZ`

### Problem: "Already a member"

**Solution:**
- You're already in the project!
- Check the project dropdown in sidebar
- Select the project to view it

### Problem: "Can't find Join Project button"

**Solution:**
- Make sure you're logged in
- Go to **Projects** page (sidebar)
- Button is at the top-right corner
- If you see "No Projects Yet", the button is in the center

## 🎓 Best Practices

### For Team Leads:

✅ **Share code immediately** after creating project
✅ **Use a group chat** to share with everyone at once
✅ **Keep code accessible** (pin message, save in docs)
✅ **Verify members joined** by checking team list
✅ **Welcome new members** in activity log

### For Team Members:

✅ **Save the invite code** when you receive it
✅ **Register immediately** to avoid delays
✅ **Use your real name** for easy identification
✅ **Select correct role** (Developer/Tester/Admin)
✅ **Check notifications** after joining

## 📊 Team Roles Explained

| Role | Permissions | Best For |
|------|-------------|----------|
| **Admin** | Full access, can manage project | Team leads, project managers |
| **Developer** | Create tasks, fix bugs, update status | Programmers, engineers |
| **Tester** | Report bugs, test features, update status | QA testers, reviewers |

## 💡 Tips

1. **Create a Welcome Message**
   - After members join, create a task titled "Welcome!"
   - Assign it to new members
   - Include project guidelines and resources

2. **Set Up Team Roles**
   - Assign tasks based on member roles
   - Developers get coding tasks
   - Testers get testing tasks

3. **Use Activity Log**
   - Monitor when members join
   - Track team activity
   - See who's most active

4. **Regular Check-ins**
   - Review team member list weekly
   - Ensure everyone has tasks assigned
   - Check if anyone needs help
