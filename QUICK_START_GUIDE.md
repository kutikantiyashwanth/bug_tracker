# Quick Start Guide

## 1. Open the App
https://bug-tracker-ui-evqv.onrender.com

## 2. Login
Use demo account or register:
- admin@test.com / password123

## 3. Create a Project (Admin)
- Go to Projects
- Click New Project
- Enter name and description
- Copy the invite code to share with team

## 4. Join a Project (Developer/Tester)
- Go to Projects
- Click Join Project
- Enter the invite code from your Admin

## 5. Report a Bug
- Go to Bug Reports
- Click Report Bug
- Fill in details
- Use AI Analyze to auto-detect severity

## 6. Manage Tasks
- Go to Kanban Board
- Create tasks with + button
- Drag cards between columns

## 7. View Analytics
- Go to Overview (Dashboard)
- See charts, completion rates, deadlines

## 8. Real-time Collaboration
- Multiple users can be logged in simultaneously
- Task moves and notifications update instantly
