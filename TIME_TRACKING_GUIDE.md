# Time Tracking Guide

## Overview
Time tracking allows you to log hours spent on tasks and bugs.

## How to Use

### Start a Timer
1. Go to **Time Tracking** in the sidebar
2. Click **Start Timer**
3. Optionally select a task or bug
4. Add a description
5. Timer starts counting

### Stop a Timer
1. Click **Stop Timer**
2. Time is automatically logged

### Log Time Manually
1. Go to Time Tracking
2. Click **Log Time**
3. Enter hours and minutes
4. Select task or bug
5. Add description
6. Save

## View Time Reports

### By Task
See total time spent on each task

### By Bug
See total time spent fixing each bug

### By User
See how much time each team member has logged

### By Project
See total project time across all members

## Notes
- Time entries are stored locally in the browser (localStorage)
- They persist across sessions on the same device
- Time tracking data is not synced to the database in the current version
