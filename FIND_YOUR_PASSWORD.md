## Forgot Your Password?

Currently there is no password reset flow in the UI.

To reset a password manually via the database:

```bash
"C:\Program Files\PostgreSQL\16\bin\psql.exe" -U postgres -d student_bug_tracker
```

Then run:
```sql
-- Find user
SELECT id, email FROM users WHERE email = 'your@email.com';

-- The password hash for 'password123' is:
-- $2a$10$... (generated by bcrypt)
-- Use the app's register endpoint to create a new account instead
```
